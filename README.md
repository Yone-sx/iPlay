# iPlay

iPlay是一款提供给初学者学习的简易音乐播放器，由本地播放、在线播放和歌曲下载三个模块组成，拥有简洁的界面和基本的播放器功能，支持上下首、设置播放模式和下载歌曲。欢迎初学者下载！
